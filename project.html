My work

2048 game using AI

The 2048 game is a single-player game. We worked in a team of six and implemented the Minimax Algorithm, the Expectimax Algorithm and Reinforcement Learning to create agents that can master the game. I created the agent using reinforcement learning. Essentially, the project involves the implementation and comparative analysis of the three algorithms.
Implementation language: Python
Libraries: tkinter, keras, numpy, random
Implementation details: https://github.com/devyani611/Artificial-Intelligence

Wine quality prediction

Implemented random forest and decision tree algorithms in machine learning to predict the quality of wine. The goal of the project was to determine white wine quality based on its chemical properties. Quality is graded from 0 to 10, where 0 is the worst and 10 is the best quality. Based on the quality scores we classified wines as Good, Bad and Normal ones. Essentially, the accuracy of random forest algorithm and decision tree algorithm had been compared.
Implementation language: Python
Libraries: sklearn, numpy, seaborn, matplotlib, graphviz, pandas.
Implementation details: https://github.com/devyani611/Machine-Learning

Database Implementation of IPL(Indian Premiere League)

The domain of this database was based on Indian Premier League (IPL). IPL, is a professional Twenty20 cricket league in India, which is contested during April and May of every year by 8 teams representing Indian cities and some states.Each team plays each other twice in a home-and-away round-robin format. It is the most-attended cricket league in the world. The data for a few tables was downloaded online, while some of the tables were populated by me.
Database: PostgreSQL
Implementation details: https://github.com/devyani611/Introduction-to-DBMS

Analysis of QuickSort, RadixSort, TimSort

Implemented QuickSort with naive pivot selection, random pivot selection and median of medians with group 5 and group 7, Radix sort and Tim sort to compare and analyze their performance. To scrutinize the algorithms thoroughly, different input lists are generated as: Unique random integers list, Few unique random integers list(may contain duplicate integers), Nearly sorted (first half sorted, second half random), Nearly sorted( second half sorted, first half random),Reverse integers list (sorted in decreasing order).
Implementation Language: Python
Libraries: random
Implementation details: https://github.com/devyani611/Analysis-and-Design-of-Algorithm
